fórmulas (uso recursivo):
a - b = temp
a - c = temp
b - c = temp

temp = armazenamento temporário

se temp for igual a zero, então 1 deverá ser incrementado na posição 132

Informações de endereçamento:
Endereço 128 -> lado A
Endereço 129 -> lado B
Endereço 130 -> lado C
Endereço 131 -> constante 1
Endereço 132 -> resultado
Endereço 250 -> cache para o valor de A

se r (end 132) igual a 0, 1 ou 3, respectivamente:
r = 0: escaleno
r = 1: isóscelez
r = 3: equilátero
